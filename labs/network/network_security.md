### 🔍 Key Learning Outcomes:
* **Traffic Analysis:** فهم كيفية عمل بروتوكولات الشبكة الأساسية (DNS, ICMP, DHCP, Kerberos).
* **Advanced Filtering:** استخدام الـ Display Filters المتقدمة لعزل الـ Packets المشبوهة بدقة.
* **Attack Detection:** كشف هجمات الـ ARP Poisoning وتحليل الـ Cleartext Credentials.
* **Decryption:** فك تشفير الـ HTTPS Traffic واستخراج الملفات المخفية.
