### 🔍 Key Learning:
* **Injection & XSS:** فهم كيفية تنفيذ الـ Malicious Scripts.
* **Access Control:** كيف يقوم المخترقون بتجاوز الصلاحيات (Authorization).
* **Data Exposure:** أساسيات حماية بيانات المستخدمين والتشفير.
